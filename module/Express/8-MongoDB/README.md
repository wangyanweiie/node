# MongoDB

[绿色版.zip](https://fastdl.mongodb.org/windows/mongodb-windows-x86_64-4.4.4.zip)

- 启动服务端

``` shell
.\mongod.exe --dbpath=E:\Project\node\module\Express\8-MongoDB\db
```

- 启动客户端

``` shell
.\mongo.exe
```

- 注意事项

``` shell
# 运行环境
node version 14.17.0

# 自己创建的 myapp 运行报错
```
